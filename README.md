# ♻️ Scrap Dealing Business Website

A modern and responsive website developed for a scrap dealing business.
This project is built using **HTML, CSS, and JavaScript** and is designed to showcase services, improve online presence, and generate client leads.

---

## 🚀 Features

* ✅ Responsive design (Mobile + Desktop)
* ✅ Clean and professional UI
* ✅ Service showcase section
* ✅ Contact form integration (Web3Forms)
* ✅ Fast and lightweight
* ✅ SEO-friendly structure

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Web3Forms (for form handling)
